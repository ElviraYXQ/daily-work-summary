## ⚡ 特别关注：免费活动咨询/投诉（627条，9.6%）

**活动机制**: 每小时随机抽取幸运订单免单

**核心问题**: 用户对规则理解不清

### 典型案例

**案例1: 误解规则（以为所有订单免费）**
```
"They have shown as a free sale but when I ordered it is shown as payable"
```

**案例2: 不知如何参与**
```
"I want to use free day sale"
"how to place a free order?"
```

**案例3: 觉得不公平**
```
"the free clothes are only for the influencers"
"They're just fooling us"
```

### 规模分析

| 时段 | 免费活动进线 | 占该时段大促进线比例 |
|------|------------|------------------|
| 04:00 | 89 | 6.7% |
| 05:00 | 460 | 7.1% |
| 06:00 | 78 | 5.1% |

**总计**: 627条（9.6%）

### 影响评估

**正面**:
- 活动吸引力强，用户参与度高

**负面**:
- 规则不清晰导致用户投诉
- 部分用户认为"只有网红能拿到"，产生不公平感
- 负面情绪可能扩散（社交媒体）

### 建议优化方案

#### P0 - 立即行动

1. **透明化规则**
   - 在活动页面醒目位置说明"每小时随机抽奖"
   - 明确标注"非全场免费"
   - 展示中奖概率（如：每小时X单）

2. **实时公布中奖名单**
   - 每小时公布免单订单号（脱敏）
   - 增加公平性和可信度
   - 刺激用户持续参与

3. **客服话术统一**
   ```
   标准回复：
   "感谢您参与522免单活动！本活动为【每小时随机抽取】幸运订单免单，
   并非全场免费。中奖用户会收到APP通知。本小时抽奖将在XX:XX公布，
   祝您好运！"
   ```

#### P1 - 中期优化

4. **增加参与感**
   - 显示"您已参与本小时抽奖"
   - 倒计时显示"距离开奖还有XX分钟"
   - 未中奖用户发送"下小时继续加油"提示

5. **降低负面情绪**
   - 为参与但未中奖用户发放小额优惠券
   - 文案调整：避免使用"免费"，改为"抽奖免单"
