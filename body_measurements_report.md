## 一、核心发现

### 🎯 关键洞察（可直接行动）

**1. 用户整体身材画像**

最主流的 3 种身材组合（占 62.5%）:
- **average臀_average腹**: 33.7% (267,603人) - 标准身材
- **average臀_flatter腹**: 16.1% (127,778人) - 标准臀型，平坦腹部
- **wider臀_curvier腹**: 12.7% (101,124人) - 大臀围，圆润腹部

**行动建议**: 产品设计和尺码表应优先覆盖这 3 种身材类型

---

**2. 尺码分布呈现极端分化**

腰围分布:
```
XXL腰(34+)    31.1% (247,271人) ← 最大群体
M腰(28-30)    20.4% (161,926人)
L腰(30-32)    15.1% (120,091人)
XL腰(32-34)   11.9% (94,571人)
S腰(26-28)    12.4% (98,166人)
XS腰(<26)      9.1% (72,331人)
```

上胸围分布:
```
XXL胸(38+)    26.8% (212,478人) ← 最大群体
M胸(32-34)    21.5% (170,640人)
L胸(34-36)    20.4% (162,181人)
XS胸(<30)     11.5% (91,691人)
XL胸(36-38)   11.1% (88,555人)
S胸(30-32)     8.7% (68,811人)
```

**⚠️ 关键发现**:
- XXL 尺码是最大用户群体（腰围 31.1%，胸围 26.8%）
- 标准尺码（S/M/L）合计仅占约 48%

**行动建议**: 尺码表和库存配比必须向大码倾斜

### 🇮🇶 伊拉克站（IQ）- 10.9 万用户

**身材特征**:
- 主流: average臀_average腹 (26.3%)，但分布更分散
- 平均腰围: **66.0cm**
- 平均上胸围: **71.3cm**
- XXL 占比: 腰围 85.3%，胸围 80.9%

**尺码分布**:
| 尺码 | 腰围占比 | 胸围占比 |
|------|---------|---------|
| XS | 7.3% | 11.1% |
| S | 1.4% | 2.0% |
| M | 1.6% | 1.9% |
| L | 2.6% | 2.1% |
| XL | 1.7% | 2.0% |
| XXL | 85.3% | 80.9% |

**⚠️ 数据质量警告**:
- 腰围 66cm、胸围 71cm **不符合常理**（正常成年女性应为 60-80cm）
- 可能原因:
  1. **单位错误**: inch vs cm（26 inch = 66cm）
  2. **测量方法**: 可能是全围而非半围
  3. **数据录入错误**

**紧急行动**:
🚨 **立即核查 IQ 站点数据采集方式和单位**
- 如果是单位问题，实际应为: 腰围 26 inch，胸围 28 inch
- 如果数据正确，说明 IQ 用户确实极度偏大码

**产品策略**（数据核查后）:
- XXL/XXXL 码为绝对主力
- 需要专门的大码版型设计
- 标准尺码仅作小量备货

## 附录: 数据说明

**字段定义**:
- `USER_HIP`: 臀围类型（wider/average/straighter）
- `USER_BELLY`: 腹部类型（curvier/average/flatter）
- `USER_WAIST`: 腰围数值（cm）
- `USER_UPPER_BUST`: 上胸围数值（cm）
- `usr_id`: 该三围组合的用户数量

**尺码分段标准**:
- XS腰: <26cm | XS胸: <30cm
- S腰: 26-28cm | S胸: 30-32cm
- M腰: 28-30cm | M胸: 32-34cm
- L腰: 30-32cm | L胸: 34-36cm
- XL腰: 32-34cm | XL胸: 36-38cm
- XXL腰: 34+cm | XXL胸: 38+cm

**分析方法**:
- 使用加权平均（按用户数量加权）
- 将臀围、腹部、腰围、胸围作为整体组合分析
- 按站点分别统计
