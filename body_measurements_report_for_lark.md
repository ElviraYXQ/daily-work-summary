**分析时间**: 2026-04-22

**目标站点**: IN (印度) / IQ (伊拉克)

---

## 📊 执行摘要

本报告基于 52,809 条用户三围数据，经过严格清洗后保留 34,084 条有效数据（清洗率 35.46%），为 IN 和 IQ 两个站点提供标准人台设计方案和版型策略建议。

### 核心发现

- **IN 站点**: 体型较集中，主导轻曲线型（55.7%），风险可控
- **IQ 站点**: 体型分散度高，需要双人台策略，高风险预警

## 5️⃣ 🎯 核心输出：人台设计方案

### IN 站点人台方案

#### 1️⃣ 标准人台 (Core Fit Model)

| 参数 | 数值 |
|------|------|
| 建议腰围 | **76.2 cm** |
| 建议胸围 | **86.4 cm** |
| 胸腰差 | **10.2 cm** |
| 对应体型 | 轻曲线型 |
| 覆盖人群 | P50-P70 区间（约 4,891 人，28.6%） |

#### 2️⃣ 次人台

❌ **不需要**

**原因**: 曲线型（标准+强曲线）用户占比仅 11.8%，不足以支撑单独人台

#### 3️⃣ 版型策略

✅ **建议以轻曲线版型为主设计**

- 适度腰部收紧（收腰 10-12cm）
- 保持舒适度，不过度贴身
- 适合日常通勤风格
- 避免过于宽松的直筒设计

#### 4️⃣ 风险评估

**体型分散度**:
- 腰围 P10-P90 跨度: 20.3 cm
- 胸围 P10-P90 跨度: 20.3 cm

✅ **风险可控**
- 体型较集中，标准人台可覆盖大多数用户
- 预计 Fitting 退货率: **< 10%**
- 建议尺码数量: **5 个**（XS, S, M, L, XL）

---

### IQ 站点人台方案

#### 1️⃣ 标准人台 (Core Fit Model)

| 参数 | 数值 |
|------|------|
| 建议腰围 | **80.0 cm** |
| 建议胸围 | **96.0 cm** |
| 胸腰差 | **16.0 cm** |
| 对应体型 | 轻曲线型 |
| 覆盖人群 | P50-P70 区间（约 3,525 人，20.8%） |

#### 2️⃣ 次人台 (Secondary Fit Model - 曲线型)

⚠️ **强烈建议设置**

| 参数 | 数值 |
|------|------|
| 建议腰围 | **73.0 cm** |
| 建议胸围 | **97.0 cm** |
| 胸腰差 | **24.0 cm** |
| 目标人群 | 曲线型用户（占比 33.0%） |

**理由**:
- 曲线型用户占比达 33%，是重要细分市场
- 腰细胸丰满，标准人台无法满足
- 可显著降低此类用户的 Fitting Issue

#### 3️⃣ 版型策略

✅ **建议双版型策略**

**主推版型（Regular Fit - 轻曲线）**:
- 覆盖 67% 直筒+轻曲线用户
- 适度收腰（12-16cm）
- 宽松舒适

**辅助版型（Slim Fit - 曲线）**:
- 覆盖 33% 曲线型用户
- 强收腰（18-25cm）
- 腰部更贴身，胸部留余量

#### 4️⃣ 风险评估

**体型分散度**:
- 腰围 P10-P90 跨度: **35.0 cm**
- 胸围 P10-P90 跨度: **35.0 cm**

⚠️ **高风险警告**

**风险**:
- 体型分散度高，单一人台难以覆盖所有用户
- 若仅使用单一版型：
  - **Fitting Issue**: 腰部过紧或过松
  - **胸围不合适**: 上身紧绷或空荡
  - **退货率上升**: 预计影响 **15-25%**
- 受影响品类: **上衣、连衣裙、外套**

**缓解措施**:
- ✅ 提供 **2 个版型**（Regular + Slim）
- ✅ 增加尺码密度: **7 个尺码**（XS, S, M, L, XL, XXL, XXXL）
- ✅ 或提供 **Petite / Regular / Plus** 系列

- **Regular Fit**: 标准版型，宽松舒适
- **Slim Fit**: 修身版型，贴身剪裁
- **Plus Size**: 大码系列（通常 XL+）
- **Petite**: 娇小系列（通常 XS-S）
